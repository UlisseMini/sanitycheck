# Option F: Property-Linking Alarm Examples

## Strategy
Flag any sentence that links two properties (X is Y because it's Z). The link X↔Z is doing the work. Is that link justified in this domain, or just asserted?

```

"[Quote: X is Y because it's Z]"

- Y (claimed property): [what is being claimed about X]
- Z (justification): [the reason given]

**Link being claimed:**
"[Z → Y]"

**Is this link justified?**
[Yes/No + why or why not in this domain]

**What's missing:**
[What evidence/reasoning would be needed]

**How to spot it:**
[What signals should trigger this check]
```
